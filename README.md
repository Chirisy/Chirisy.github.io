# Chirisy 的个人网站

> 记录技术学习、项目实践和长期积累的个人知识库。

## 关于这里

这个网站使用 [docsify](https://docsify.js.org/) 搭建，内容以 Markdown 文件维护，适合持续更新技术笔记、教程、项目文档和个人档案。

当前主要内容包括：

- 技术笔记：沉淀编程、工程实践、AI 与系统相关内容
- 项目实践：整理做过的项目、实验和复盘
- VERL 使用教程：围绕大语言模型强化学习训练框架 verl 的实践文档
- 关于我：放置个人介绍、联系方式和常用链接

## 快速入口

## 本地预览

在仓库根目录启动任意静态服务器即可预览：

```bash
node serve.js
```

然后访问：

```text
http://localhost:3000
```

## 部署

这个仓库已经包含 `.nojekyll`，可以直接作为 GitHub Pages 站点部署。推送到 `main` 分支后，在 GitHub 仓库的 `Settings -> Pages` 中选择从分支部署即可。
